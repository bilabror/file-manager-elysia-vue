# Example File Manager with Vue 3 and Eysia JS

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/eysia/eysia-file-manager.git
```

2. Setup database

```bash
bun docker:up
```

3. Install dependencies

```bash
bun i
```

1. Run the development (backend and frontend)

```bash
bun dev
```
